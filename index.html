JS. Animals hierarchy

Необходимо создать иерархию классов животных:

Animals => {Mammals, Birds, Fish} => {Mammals => Predators, Whales, Primates} =>

{{Predators => Dog},

{Whales => Dolphin},

{Primates => Human}​}

Определить три метода: move(), say() and eat() в классе Animals. Переопределить эти методы для Birds, Fish, Dog, Dolphin и Human.

У этих классов должны быть:
- собственные конструкторы,
- поля: age, name, weight, limbAmount (количество конечностей),
- публичные статические поля: maxAge, maxWeight,
- статические методы: is[className] => boolean.

Создать по 2-а объекта каждого класса. Протестировать эти объекты и их методы: вывести в консоль результаты работы методов, получение свойств.
